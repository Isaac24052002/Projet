# prix_maison

## Aperçu
Régression de prix de maisons avec comparaison de plusieurs familles de modèles.

## Fichier principal
- `prix_maison.ipynb`

## Modèles utilisés
- `LinearRegression`
- `RandomForestRegressor`
- `XGBRegressor`
- `LGBMRegressor`
- Réseau de neurones (`Sequential`, `Dense`, `Dropout`)

## Résultats
- Les sorties montrent les modèles entraînés mais pas de tableau final clair (`RMSE/MAE/R²`) dans la version sauvegardée.

- Ajouter un résumé métriques final par modèle améliorera la lisibilité.
