# titanic(2)

## Aperçu
Classification de survie des passagers Titanic.

## Fichier principal
- `titanic(2).ipynb`

## Modèles utilisés
- `LogisticRegression`
- `SVC`
- `RandomForestClassifier`

- Après tuning RandomForest, une sortie affiche aussi `accuracy = 0.78`
