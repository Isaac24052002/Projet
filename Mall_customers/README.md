# Mall_customers

## Aperçu
Segmentation client sur données de centre commercial.

## Fichier principal
- `Mall_customers.ipynb`

## Modèles utilisés
- `KMeans`
- `PCA` (visualisation/réduction de dimension)

## Résultats
- Pas d'`accuracy` attendue (clustering non supervisé).
- Les métriques utiles à documenter sont plutôt `silhouette score` et inertie.
