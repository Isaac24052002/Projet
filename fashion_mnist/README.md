# fashion_mnist

## Aperçu
Classification d'images Fashion-MNIST avec CNN.

## Fichier principal
- `fashion_mnist.ipynb`

## Modèles utilisés
- CNN Keras (`Conv2D`, `MaxPooling2D`, `Flatten`, `Dense`, `Dropout`, `Sequential`)

## Résultats (sorties sauvegardées)
- `accuracy` test: environ `0.90`
- `val_accuracy` observée pendant entraînement: jusqu'à environ `0.90`

## Notes
- Le notebook contient aussi un rapport `precision/recall/f1-score`.
