# Immobilier

## Aperçu
Régression de prix immobiliers avec comparaison de plusieurs modèles.

## Fichier principal
- `Immobilier.ipynb`

## Modèles utilisés
- `LinearRegression`
- `KNeighborsRegressor`
- `DecisionTreeRegressor`
- `RandomForestRegressor`
- `GradientBoostingRegressor`
- `SVR`

## Résultats (sorties sauvegardées)
- Meilleur modèle indiqué dans le notebook: `SVR`
- `RMSE`: `24659.58`

## Notes
- En régression, on suit surtout `RMSE`, `MAE` et `R²` (pas l'accuracy).
